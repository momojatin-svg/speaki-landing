# Speaki Landing Page

This repository contains a responsive, premium SaaS-style landing page built with:

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Sticky, centered navigation bar with glassmorphism styling
- Premium dark hero section with gradient text and CTA buttons
- Three-device hero mockup layout
- Feature strip with hover states
- Waitlist panel with responsive form layout
- Footer with social icons
- Fully responsive design for mobile, tablet, and desktop

## Setup

1. Open `index.html` in your browser.
2. No build step or dependencies are required.

## Files

- `index.html` — page markup
- `styles.css` — design system and page styles
- `script.js` — form interaction logic

## Notes

- The page is designed to closely match the provided screenshot.
- The `hero-visual` uses modern CSS layout and shadowing for the device mockups.
